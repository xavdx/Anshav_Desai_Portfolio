001
Food Image Classification

Designed a complete computer vision pipeline for 10-class food image classification using PyTorch, with stratified splits, image augmentation, and ImageNet normalization. Fine-tuned ResNet-50, ResNet-101, and EfficientNet-B0 with AdamW and cosine scheduling. Interpreted results via confusion matrices and Grad-CAM visualizations, exporting the best model via TorchScript.

~0.94 F1 score

Built an end-to-end binary image classification system (Bird vs Drone) using a Custom CNN and EfficientNetB0 with TensorFlow/Keras. Tackled class imbalance, fine-tuned the model, and deployed a Streamlit web application for real-time inference with confidence scores — cloud-hosted and version-controlled on GitHub.

98.6% accuracy
